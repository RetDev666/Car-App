# Car Manager - Система управління автомобілем

Веб-додаток для управління автомобілем з красивим сучасним дизайном та повним функціоналом для відстеження витрат, технічного обслуговування та планування поїздок.

## Функціонал

- 🚗 **Управління автомобілями** - додавання, редагування та видалення автомобілів
- ⛽ **Журнал витрат на паливо** - відстеження заправок та розрахунок витрати палива
- 🔧 **Технічне обслуговування** - ведення історії ТО та нагадування
- 💰 **Відстеження витрат** - журнал всіх витрат (паливо, ремонт, страховка)
- 🏪 **База автосервісів** - контакти сервісів з рейтингами
- 🗺️ **Планувальник поїздок** - розрахунок вартості поїздок
- 📊 **Калькулятор витрат** - розрахунок загальної вартості утримання авто
- 🔔 **Нагадування** - автоматичні нагадування про ТО та страховку

## Технології

- **Frontend**: HTML5, CSS3, Bootstrap 5, JavaScript (ES6+)
- **Backend**: Python Flask
- **База даних**: SQLite
- **Іконки**: Font Awesome 6
- **Графіки**: Chart.js

## Встановлення

### Вимоги
- Python 3.7+
- pip (менеджер пакетів Python)

### Швидкий запуск

1. **Запустіть тестову версію (рекомендовано)**
   ```bash
   # Windows
   start_test.bat
   
   # Або вручну
   python run.py
   ```

2. **Відкрийте браузер**
   - Повна версія: `http://localhost:5000`
   - Тестова версія: відкрийте `test.html` в браузері

### Ручне встановлення

1. **Клонуйте репозиторій**
   ```bash
   git clone <repository-url>
   cd Car-App
   ```

2. **Встановіть залежності**
   ```bash
   pip install -r requirements.txt
   ```

3. **Створіть демонстраційні дані**
   ```bash
   cd backend
   python demo_data.py
   ```

4. **Запустіть додаток**
   ```bash
   python app.py
   ```

5. **Відкрийте браузер**
   Перейдіть за адресою: `http://localhost:5000`

## Використання

### Додавання автомобіля
1. Натисніть кнопку "Додати авто" на головній сторінці
2. Заповніть інформацію про автомобіль
3. Натисніть "Додати"

### Ведення журналу палива
1. Перейдіть до розділу "Журнал палива"
2. Натисніть "Додати заправку"
3. Вкажіть кількість палива, ціну та пробіг
4. Система автоматично розрахує витрату палива

### Додавання витрат
1. Перейдіть до розділу "Витрати"
2. Натисніть "Додати витрату"
3. Оберіть тип витрати та вкажіть суму
4. Витрати будуть відображені в статистиці

### Планування поїздок
1. Перейдіть до розділу "Планувальник поїздок"
2. Натисніть "Планувати поїздку"
3. Вкажіть маршрут та відстань
4. Система розрахує вартість поїздки

## API Endpoints

### Автомобілі
- `GET /api/vehicles` - отримати всі автомобілі
- `POST /api/vehicles` - додати новий автомобіль
- `PUT /api/vehicles/<id>` - оновити автомобіль
- `DELETE /api/vehicles/<id>` - видалити автомобіль

### Витрати
- `GET /api/expenses` - отримати всі витрати
- `POST /api/expenses` - додати нову витрату

### Паливо
- `GET /api/fuel-logs` - отримати журнал палива
- `POST /api/fuel-logs` - додати заправку

### ТО
- `GET /api/maintenance` - отримати записи ТО
- `POST /api/maintenance` - додати запис ТО

### Сервіси
- `GET /api/services` - отримати всі сервіси
- `POST /api/services` - додати новий сервіс

### Поїздки
- `GET /api/trips` - отримати всі поїздки
- `POST /api/trips` - додати нову поїздку

### Статистика
- `GET /api/stats` - отримати статистику дашборду

## 🗄️ База даних

### Структура бази даних
Додаток використовує SQLite базу даних з повною схемою:

- **8 основних таблиць** з повними зв'язками
- **Індекси для продуктивності** 
- **Обмеження цілісності** (foreign keys, check constraints)
- **Представлення (views)** для статистики
- **Тригери** для автоматичних оновлень

### Демонстраційні дані
База містить готові тестові дані:
- 5 автомобілів різних марок
- 50 записів заправок
- 25 записів ТО
- 100 витрат різних категорій
- 5 автосервісів з рейтингами
- 40 поїздок з розрахунками
- 5 страхових полісів
- 15 нагадувань
- 150 записів цін на паливо

### Файли бази даних
- `database/schema.sql` - SQL схема бази даних
- `database/init_db.py` - Скрипт ініціалізації
- `database/README.md` - Документація бази даних
- `car_manager.db` - SQLite файл бази даних (створюється автоматично)
