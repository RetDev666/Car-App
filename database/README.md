# Car Manager Database

База даних для додатку управління автомобілем з повною схемою та демонстраційними даними.

## 📊 Структура бази даних

### Основні таблиці

#### 🚗 `vehicles` - Автомобілі
- `id` - Унікальний ідентифікатор
- `brand` - Марка автомобіля
- `model` - Модель автомобіля
- `year` - Рік випуску
- `plate` - Номерний знак (унікальний)
- `mileage` - Поточний пробіг
- `color` - Колір
- `engine_type` - Тип двигуна
- `fuel_capacity` - Об'єм паливного бака
- `created_at` - Дата створення
- `updated_at` - Дата оновлення

#### ⛽ `fuel_logs` - Журнал палива
- `id` - Унікальний ідентифікатор
- `vehicle_id` - ID автомобіля (зв'язок)
- `date` - Дата заправки
- `amount` - Кількість палива (л)
- `price_per_liter` - Ціна за літр
- `total_cost` - Загальна вартість
- `mileage` - Пробіг на момент заправки
- `station_name` - Назва АЗС
- `fuel_type` - Тип палива
- `notes` - Примітки

#### 🔧 `maintenance_records` - Технічне обслуговування
- `id` - Унікальний ідентифікатор
- `vehicle_id` - ID автомобіля (зв'язок)
- `date` - Дата ТО
- `type` - Тип робіт
- `mileage` - Пробіг на момент ТО
- `cost` - Вартість робіт
- `description` - Опис робіт
- `service_center` - Назва сервісу
- `next_maintenance_date` - Дата наступного ТО

#### 💰 `expenses` - Витрати
- `id` - Унікальний ідентифікатор
- `vehicle_id` - ID автомобіля (зв'язок)
- `date` - Дата витрати
- `type` - Тип витрати (fuel, maintenance, insurance, repair, other)
- `description` - Опис витрати

#### 🏢 `services` - Автосервіси

- `name` - Назва сервісу
- `address` - Адреса
- `phone` - Телефон
- `email` - Email
- `website` - Веб-сайт
- `rating` - Рейтинг (1-5)
- `description` - Опис послуг
- `services_offered` - Перелік послуг
- `working_hours` - Робочі години
- `is_active` - Активний статус

#### 🛣️ `trips` - Поїздки
- `id` - Унікальний ідентифікатор
- `vehicle_id` - ID автомобіля (зв'язок)
- `date` - Дата поїздки
- `from_location` - Початкова точка
- `to_location` - Кінцева точка
- `distance` - Відстань (км)
- `fuel_price` - Ціна палива
- `estimated_cost` - Розрахункова вартість
- `actual_cost` - Фактична вартість
- `purpose` - Мета поїздки

#### 🛡️ `insurance_records` - Страховка

#### 🔔 `reminders` - Нагадування

#### ⛽ `fuel_prices` - Ціни на паливо

- `fuel_type` - Тип палива

- `station_name` - Назва АЗС
- `location` - Місцезнаходження
- `date` - Дата
- `created_at` - Дата створення

Агрегована статистика по кожному автомобілю:

- Загальна вартість палива

## 🔧 Технічні особливості

### Індекси для продуктивності
- Індекси по `vehicle_id` для всіх пов'язаних таблиць
- Індекси по датах для швидкого пошуку
- Індекси по типах витрат
- Індекси по статусах нагадувань

### Обмеження цілісності
- Зовнішні ключі між всіма пов'язаними таблицями
- Перевірка значень (CHECK constraints)
- Унікальні обмеження для номерних знаків
- Обмеження на рейтинги (1-5)

### Тригери
- Автоматичне оновлення `updated_at` при зміні записів
- Обчислення `total_cost` для заправок

## 📊 Приклади запитів

### Найпопулярніший автомобіль за витратами
```sql
SELECT v.brand, v.model, COUNT(e.id) as expense_count
FROM vehicles v
LEFT JOIN expenses e ON v.id = e.vehicle_id
GROUP BY v.id, v.brand, v.model
ORDER BY expense_count DESC
LIMIT 1;
```

### Середня витрата палива
```sql
SELECT AVG((f.amount / (f.mileage - LAG(f.mileage) OVER (PARTITION BY f.vehicle_id ORDER BY f.date)) * 100)) as avg_consumption
FROM fuel_logs f
WHERE f.mileage > LAG(f.mileage) OVER (PARTITION BY f.vehicle_id ORDER BY f.date);
```

### Витрати по місяцях
```sql
SELECT strftime('%Y-%m', date) as month, SUM(amount) as total
FROM expenses
GROUP BY strftime('%Y-%m', date)
ORDER BY month DESC;
```
