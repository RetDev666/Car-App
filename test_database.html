🚀 Інструкції по запуску
1. Ініціалізація бази даних:
python database/init_db.py
2. Запуск Flask додатку:
python run_with_db.py
3. Альтернативно (Windows):
start_with_db.bat
4. Тестування бази даних:
python test_database.py
Примітка: Переконайтеся, що Python встановлений і доступний в PATH.
