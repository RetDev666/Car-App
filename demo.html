Демонстраційна версія

Це демо версія додатку Car Manager з демонстраційними даними. Всі функції працюють локально без сервера.

Для повної версії з базою даних:
1. Встановіть Python
2. Запустіть python database/init_db.py
3. Запустіть python run_with_db.py
